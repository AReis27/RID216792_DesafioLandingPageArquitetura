Landing Page de Arquitetura

Projeto desenvolvido como desafio da Escola DNC com o objetivo de criar uma landing page utilizando HTML, CSS e JavaScript, além de integrar um formulário com o SheetMonkey e hospedar o site na internet.

- Tecnologias utilizadas

HTML5

CSS3

JavaScript

SheetMonkey

Netlify

GitHub

- Objetivo do projeto

Desenvolver uma landing page para uma empresa de arquitetura, baseada no layout fornecido no Figma, aplicando conceitos de desenvolvimento web e integração de formulários.

- Funcionalidades

Layout responsivo baseado no Figma

Seção institucional da empresa

Estatísticas de projetos realizados

Formulário de contato

Integração com planilha via SheetMonkey
